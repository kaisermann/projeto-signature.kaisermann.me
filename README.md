# signature.kaisermann.me

> Personal centralized website signature helper script.

## Usage: 

```
  <script src="https://signature.kaisermann.me" data-style="
  position: xxxx; 
  bottom|right|top|left: xxxx;
  color: xxxx;
  background-color: xxxxx;
  "></script>
```
